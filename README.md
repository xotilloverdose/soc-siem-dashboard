# SOC SIEM Dashboard

A SOC-style Security Information and Event Management dashboard built with Flask.

## Features

* Live log monitoring
* Threat scoring
* Rule-based detection
* Quarantine / unban workflow
* Alert history
* Login authentication
* Attack map visualization
* REST API support
* SQLite storage
* Incident reporting

## Technologies

* Python
* Flask
* Flask-SocketIO
* SQLite
* HTML/CSS
* Requests

## Run

```bash
pip install -r requirements.txt
python app.py
```
